# Donation Frontend (React + Vite + Tailwind)

## Quick start
1. Extract the ZIP
2. `cd donation-frontend-full`
3. `npm install`
4. `npm run dev`
5. Open http://localhost:5173 and ensure backend is running on http://localhost:3000

This frontend includes pages: Dashboard, Donors, Donations, Products, Inventory, Disasters, Orphanages and QR generator/scanner.
